# Portafolio-Web-P-blico
Sitio Web de dominio público para la publicación de portafolios y sus respectivos proyectos.

**📌 Descripción:**
<br>
Plataforma web de portafolios desarrollada con Flask que permite a los usuarios crear, visualizar y gestionar perfiles profesionales dinámicos. El sistema almacena información en una base de datos, integra contenido externo mediante la API de GitHub y automatiza la generación de proyectos asociados a cada usuario. Además, incorpora filtrado por habilidades y recomendación de perfiles, facilitando la exploración y conexión entre desarrolladores. 
<br>

**🔧 Tecnologías:** Python, SQLite, Uuid, Flask, Requests, Werkzeug, Jinja2, HTML, CSS, GitHub API
<br>

**📝 Funciones:**
<br>
<ul>
  <li>Gestión de base de datos relacional.</li>
  <li>Manipulación de filtros.</li>
  <li>Renderización con Jinja2</li>
  <li>Arquitectura cliente-servidor.</li>
  <li>Normalización y parsing de datos.</li>
  <li>Optimización de consultas.</li>
  <li>Procesamiento de formularios.</li>
  <li>Generación de identificadores únicos.</li>
  <li>Manejo de errores en API.</li>
</ul>

Frontend, backend y base de datos.
